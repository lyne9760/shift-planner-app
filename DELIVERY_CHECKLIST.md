```
1. 用浏览器打开 scheduler.html
2. 查看完整的排班表和人员信息
3. 支持所有现代浏览器
```

### 方法2: 查看JSON数据
```
1. 打开 schedule_plan_2025_05.json
2. 查看排班配置和结果
3. 可用于数据集成
```

### 方法3: 导出为Excel
```bash

# 运行导出脚本

# 输出文件
排班表_2025年5月.xlsx
```
