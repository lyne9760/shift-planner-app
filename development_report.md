## 📞 使用方法

### 查看排班表
1. 打开 `scheduler.html` 在浏览器中查看
2. 或查看 `schedule_plan_2025_05.json` 中的排班数据

### 导出 Excel
```bash
# 安装依赖
pip install pandas openpyxl

# 运行导出脚本
python schedule_export.py
```

### 修改排班
1. 在 `schedule_export.py` 中修改 `duties` 列表
2. 或在 `scheduler.html` 中修改 `scheduleData` 对象
3. 重新运行生成脚本
