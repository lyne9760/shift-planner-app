## 💡 常见问题

### Q1: 如何查看完整排班？
**A**: 打开 `scheduler_v2.html` 文件，用任何浏览器打开即可。包含完整的29天排班表。

### Q2: 为什么原来的"休息日"现在也要排班？
**A**: 根据您的反馈，医院需要全年365天运营，所以原"休息日"（5-10,11等）现在也需要安排人员。

### Q3: 排班是否均衡？
**A**: 非常均衡！最多差异仅1天（有人排3班，有人排4班），这是理想的均衡状态。

### Q4: 可以修改排班吗？
**A**: 可以。您可以：
- 修改HTML文件中的表格
- 修改JSON文件并重新解析
- 运行Python脚本重新生成

### Q5: 如何与其他系统集成？
**A**: 使用 `schedule_final_v2.json` 文件。这是标准JSON格式，可直接导入到任何系统。

### Q6: 支持多个月份吗？
**A**: 目前这是5月6日-6月3日的排班。如需其他月份，可以修改脚本生成。

### Q7: 如何处理请假或调换？
**A**: 这个系统是基础排班方案。实际执行时可以根据请假申请进行临时调整。
