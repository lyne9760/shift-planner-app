# 医院排班系统 - 心胸外科ICU

## 📋 项目概述

这是一个智能医院排班系统，用于心胸外科ICU的一值、二值人员均衡排班。系统根据医院的复杂约束条件生成最优排班方案。

## 🎯 系统需求

### 需求1：均匀排班
- ✅ 平均每个人的值班总数
- ✅ 平均周末、周五、节假日的分配
- ✅ 不能过于密集的排班（最少间隔3天以上）

### 需求2：一值人员规则
- ✅ 一值需要两个人
- ✅ 每个人来自不同的小组
- ✅ 避免同一个小组一起值班
- ✅ 一值其中至少一个人是进修或重症或麻醉科的

### 需求3：二线人员排班
- ✅ 先固定好二值的（在"重症二三值班表.xlsx"中）
  - 李思：5-5, 5-15, 5-25
  - 刘芙蓉：5-10, 5-21, 5-28
  - 张宝、杨嵩、成艳美的班表保留不变
- ✅ 再排林琳、李嘉翔、杨芸、谢文礼的班

### 需求4：一值人员分组
- ✅ 把一值人员平均分配到三个小组
- ✅ 每个组的进修人员平均分配
- ✅ 重症或麻醉的人也平均分配

## 🔧 技术方案

### 核心算法
系统采用改进的贪心算法 + 本地搜索优化：

1. **初始化阶段**
   - 加载人员数据和约束条件
   - 计算有效的搭配组合

2. **排班生成阶段**
   - 对每个工作日，选择最优搭配
   - 优先选择值班次数较少的人员
   - 检查所有约束条件

3. **优化阶段**
   - 检查排班均衡性
   - 调整密度过高的人员
   - 验证周末/周五分配

4. **验证阶段**
   - 检查所有约束是否满足
   - 生成平衡分析报告

### 约束检查清单
- [ ] 一值两人来自不同小组
- [ ] 至少一人有特殊背景（进修/重症/麻醉）
- [ ] 同一小组不同时出现在一值中
- [ ] 值班间隔不少于3天
- [ ] 每人值班次数平衡（差异 ≤ 2）
- [ ] 周末/周五分配均衡

## 🔐 约束验证

系统会自动检查以下约束：

1. **小组约束**
   ```python
   if staff1.group == staff2.group:
       return False  # 不允许同小组
   ```

2. **背景约束**
   ```python
   if not (staff1.special_bg or staff2.special_bg):
       return False  # 至少一人有特殊背景
   ```

3. **密度约束**
   ```python
   if (date - last_duty_date).days < 3:
       return False  # 间隔不足3天
   ```

4. **均衡约束**
   ```python
   if duty_count > average + 2:
       return False  # 超过平均值过多
   ```
