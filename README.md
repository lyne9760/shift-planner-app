# 排班应用

这是一个面向轮班场景的排班应用示例，包含浏览器端界面、基础排班逻辑、Excel 导入导出和规则校验能力。仓库中的公开内容已经替换为通用示例数据，可作为二次开发起点。

## 功能特性

- 按周期生成双人值班安排
- 支持分组、角色、可排日期、偏好搭档等约束
- 支持导入 Excel、导出项目快照和表格结果
- 支持统计值班次数、周末覆盖和简单公平性检查

## 适用场景

- 医疗、运维、门店、客服等需要轮班的团队
- 需要离线使用、快速出表的轻量级排班场景
- 想从现有原型继续定制规则和界面的项目
