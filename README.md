# ICU 排班助手

这是一个面向 ICU 月度值班场景的本地排班工具，核心入口是浏览器端单文件页面 [ICU排班助手.html](/C:/Users/msi/Desktop/cc/排班/ICU排班助手.html)。  
它支持导入住培轮转表、上月班表和二值表，在本地完成一值/二线排班、校验、统计与 Excel 导出。

## 当前能力

- 导入当月住培轮转安排，自动提取心胸外 ICU 人员
- 导入上月班表，保留月末衔接信息和老 ICU 人员
- 导入当月二值表，自动识别二值、覆盖和假期
- 按主班/副班/二线生成整月排班
- 支持人员分组、禁排日期、搭档偏好、固定周期二线
- 支持覆盖日附加副班、周六适度放宽同组限制
- 输出矩阵班表、逐日明细、统计信息和 Excel 文件

## 排班风格

当前自动排班更偏向“接近人工排班”的风格，而不是死板追求绝对均分：

- 主班优先在较稳定的核心主班池中轮转
- 总值班次数平衡优先于部分软约束
- 搭档偏好会尽量延续，减少每天完全重新洗牌
- 月底排班会适度放宽，避免因为约束过硬而直接断排

这意味着结果更像“可用的初稿”，适合在此基础上人工微调。

## 使用方式

浏览器端：

```bash
直接打开 ICU排班助手.html
```

推荐使用顺序：

1. 在第一页导入当月住培轮转安排
2. 导入上个月班表
3. 导入当月二值表
4. 检查人员分组、禁排日期和搭档偏好
5. 生成排班并导出 Excel

## 适合的使用方式

- 先让系统生成一个接近可用的初稿
- 再按科室实际要求做人工微调
- 把手工版规律反向沉淀到下个月的自动规则里

如果你的目标是“完全自动且绝对平均”，这套工具目前不是那个方向；它更适合本地快速出初稿和辅助人工排班。
