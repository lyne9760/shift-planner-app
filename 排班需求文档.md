# 排班需求文档

## 一、基本规则

### 1.1 每日排班要求
- 每天都需要排班，无空缺日。
- 每天设三个岗位：**主班**、**副班**、**二线**。

### 1.2 岗位职责概要
| 岗位 | 说明 |
|------|------|
| 主班 | 由进修人员或重症学生担任 |
| 副班 | 与主班不同组 |
| 二线 | 部分日期不需要排（见第四条） |

---

## 二、分组约束

### 2.1 主班与副班的分组限制
- 共有 **3 个小组**。
- **主班和副班不能来自同一组**（必须跨组搭配）。

### 2.2 人员分布要求
- 主班和副班的人员需**大致平均分配**到 3 个小组中。
- **进修人员**也要尽量平均分配到各组。
- 人员名单及成分（进修/重症等）由排班负责人提供。

---

## 三、时间分配规则

### 3.1 周末与工作日均衡
- **周五、周六、周日** 与 **周一至周四** 的排班要**平均分配**，避免某人集中在周末或集中在工作日。

### 3.2 排班间隔
- 同一人的**两次排班不能隔得太近**（具体最小间隔天数待定，建议 ≥ 2 天）。
- 避免连续两天排班。

---

## 四、二线排班例外

- 存在部分日期**不需要安排二线**。
- 这些日期由排班负责人**提前指定**。
- 除指定日期外，其余日期均需安排二线。

---

## 五、个人特殊要求

- 每个人可能会有特殊排班约束（如某天不能值、只能值某类班等）。
- 特殊要求需**提前说明**，排班时统一纳入考虑。
- 建议以列表形式整理，格式如下：

```
姓名    所属小组    人员成分    特殊要求
张三    A组       进修       周五不能值
李四    B组       重症       仅值主班
...
```

---

## 六、待确认信息清单

以下信息需排班负责人补充：

1. **人员名单**：总人数、每人姓名、所属小组、人员成分（进修/重症/其他）。
2. **最小间隔**：同一人两次排班之间至少间隔几天？
3. **排班周期**：按月排还是按周排？起始日期是哪天？
4. **不排二线的日期**：具体哪些天不需要二线？
5. **个人特殊要求**：每个人的限制条件。
6. **主班候选池**：进修人员和重症学生各多少人？是否所有主班都必须从这两类人中出？

---

## 七、排班检查清单

排班完成后，按以下清单逐项核查：

- [ ] 每天三个岗位均已安排人员
- [ ] 主班和副班来自不同组
- [ ] 周五/六/日与工作日分配均衡
- [ ] 同一人无连续或过近的排班
- [ ] 各组人数大致均衡
- [ ] 进修人员均分到各组
- [ ] 指定日期无二线安排
- [ ] 所有个人特殊要求均已满足
